Place your static assets here. Vite will serve files from this folder at the root path.

Examples:
- public/assets/hero-home.jpg     ->  /assets/hero-home.jpg
- public/assets/bali.jpg          ->  /assets/bali.jpg
- public/assets/santorini.jpg     ->  /assets/santorini.jpg
- public/assets/tokyo.jpg         ->  /assets/tokyo.jpg
- public/assets/paris.jpg         ->  /assets/paris.jpg

Note: We intentionally reference placeholder URLs like /assets/hero-home.jpg in code. Add real images with these filenames to display them.
